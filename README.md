# Local Search for Docusaurus v2

Local search for Docusaurus v2.

## Installation

```bash
yarn add @cmfcmf/docusaurus-search-local
```

or

```bash
npm install @cmfcmf/docusaurus-search-local
```

## Usage

Add it to the `plugins` array in `docusaurus.config.js`:

```js
module.exports = {
  // ...
  plugins: [
    '@cmfcmf/docusaurus-search-local'
  ],
  // ...
}
```
